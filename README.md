# FamGuard

FamGuard - Account Management System built with Next.js

## Description

FamGuard is an account management system that helps manage and protect family accounts.

## Tech Stack

- Next.js 14
- React 18
- Supabase
- PostgreSQL

## Getting Started

### Prerequisites

- Node.js
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Start Production Server

```bash
npm start
```

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm start` - Start production server
- `npm run lint` - Run ESLint
- `npm run test:db` - Test database connection
